一、装置示意与动态传播（3D）

该区域使用 Three.js 展示无限深势阱的空间示意：粒子被限制在 0 到 L 的区间内，势阱两侧势能无限高，阱内允许形成驻波型定态。

二、波函数与概率密度（持续动态）

下方同时显示定态波函数 ψ(x) 和概率密度 |ψ(x)|²。波函数相位采用持续动态展示，更有“量子态不是粒子轨迹而是波函数”的感觉。

当前现象说明将在这里实时更新。

三、公式说明

势阱区间：0 < x < L
波函数：ψₙ(x)=√(2/L)·sin(nπx/L)
概率密度：|ψₙ(x)|²
能级：Eₙ ∝ n² / L²

四、参数说明

量子数 n
决定定态阶数与节点数。
阱宽 L
决定空间尺度与能级间距。
幅度缩放 A
只影响可视化显示高度，不改变物理规律。
动画速度
控制波函数相位演示速度。

五、实验过程

  1. 设定量子数 n，选择需要观察的定态。
  2. 设置阱宽 L，观察波函数空间分布变化。
  3. 观察 ψ(x) 的波形与 |ψ(x)|² 的概率分布。
  4. 比较不同 n 下节点数、波峰数与能级差异。
  5. 理解“粒子不是沿某条经典轨迹运动，而是由波函数描述状态”。

六、注意事项

  • ψ(x) 可以为正也可以为负，但 |ψ(x)|² 永远非负。
  • 节点位置处 ψ(x)=0，因此概率密度也为 0。
  • 量子数 n 越大，节点越多，能量越高。
  • 阱宽 L 越小，能级间距越大。
  • 答辩时建议重点切换 n=1、2、3 的效果进行对比展示。
