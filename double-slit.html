下方条纹区不是静止图，而是持续动态显示。光强曲线、包络线和屏幕条纹会保持动画状态，同时响应参数调节。

三、公式说明

条纹间距：Δx = λL / d
相位项：β = πdx / (λL)
双缝干涉：I(x) ∝ cos²(β)
考虑缝宽：I(x) ∝ cos²(β) · (sinα / α)²

本页考虑了单缝衍射包络，因此比理想双缝模型更接近真实实验。

四、参数说明

λ（波长）
控制颜色近似和条纹间距。
d（缝距）
控制干涉条纹疏密。
a（缝宽）
控制衍射包络宽窄。
L（屏距）
控制条纹整体舒展程度。
B（亮度）
增强条纹对比效果。
v（速度）
控制 3D 波前传播和图像动画速度。
规律总结：
λ↑ → 条纹变疏
L↑ → 条纹变疏
d↑ → 条纹变密
a↑ → 包络变窄

五、实验过程

  1. 准备单色光源，并使其照射双缝板。
  2. 双缝将光分为两列相干光波。
  3. 两列光在屏幕上叠加形成明暗条纹。
  4. 调节 λ、d、a、L 观察条纹和包络变化。
  5. 对照公式分析实验现象与参数关系。

六、注意事项

  • 光源越接近单色，条纹越清晰。
  • 缝宽、缝距与屏距共同决定最终图像形态。
  • 真实实验中存在衍射包络，不是无限均匀条纹。
  • 观察时应重点把握参数变化与现象变化的对应关系。
  • 答辩时建议现场拖动 λ、d、a 三个参数进行展示。
